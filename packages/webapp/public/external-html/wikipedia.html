JavaScript (JS) is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS.[1] As of 2022, 98% of websites use JavaScript on the client side for webpage behavior,[2] often incorporating third-party libraries.

JavaScript is a high-level, often just-in-time compiled language that conforms to the ECMAScript standard.[3] It has dynamic typing, prototype-based object-orientation, and first-class functions.

History

The first web browser with a graphical user interface, Mosaic, was released in 1993. The lead developers of Mosaic founded Netscape, which released Netscape Navigator in 1994.
