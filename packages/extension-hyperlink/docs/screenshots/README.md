# README screenshots

Generated by `bun run docs:screenshots` (see `cypress/screenshots/popovers.cy.ts`).

The PNGs in this folder are referenced from `../../README.md` via raw
GitHub URLs so npm.com renders them on the package page (relative paths
work on github.com but break on npm.com).

Excluded from the published npm tarball by `package.json#files`, which
whitelists `dist` only. Regenerate after any UI change to the prebuilt
popovers, then commit the updated PNGs.
